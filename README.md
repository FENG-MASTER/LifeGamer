# Mr.Game #

## 简介 ##
模仿LifeRPG做的一款时间管理的APP

>没想到这个创意真的有人做了APP,看来自己还是晚了一步

## 为什么要模仿 ##

因为LifeRPG并没有开源,我觉得这个开源是个不错的主意,因为基于这个创意下,点子真的太多了,需要更多的开发者加入才行


## 进度 ##

以发布1.0测试版本.

基本功能已经可以使用,但存在bug,加速修复中
